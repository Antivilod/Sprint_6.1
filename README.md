# Финальный проект 6 спринта: Яндекс.Самокат

## Описание проекта
Автотесты для учебного сервиса «Яндекс.Самокат» с использованием Selenium, Pytest и Allure.

## Структура проекта
- `src/pages/` - Page Object классы
- `src/locators/` - Локаторы элементов
- `tests/` - Тестовые классы
- `data/` - Тестовые данные
- `allure_results/` - Результаты Allure отчетов
